# 프로젝트 설치 및 실행 방법

## ✅ 설치방법

1. Node.JS를 다운받아 설치해주세요. vite를 사용하였기 때문에 최소 14.18 버전 이상이 요구됩니다.

```
https://nodejs.org/
```

<br/>

2. 리포지토리를 클론해주세요.<br/>

```
https://github.com/Poylib/pre-onboarding-3team-2nd
```

<br/>

3.dependencies를 설치해주세요.

```
npm install --force
```

<br/>
<br/>

## ✅ 실행방법

1. 다음 명령어를 이용해 Dev server를 실행시켜주세요.

```
npm run dev
```

<br/>

2. 브라우저에서 http://localhost:3000에 접속해주세요.
